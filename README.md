# vokeh

A Vue + Firebase open photo sharing social media network.

## Requirements

- Node@8

## Project setup

- `npm install`

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Run your tests

```
npm run test
```

### Lints and fixes files

```
npm run lint
```
